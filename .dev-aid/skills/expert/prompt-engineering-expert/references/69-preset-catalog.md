## Using Presets

To use a preset:
1. Select the preset number or name
2. System loads pre-configured prompt template
3. User can customize before finalizing
4. Choose output format (XML, Claude, ChatGPT, Gemini)
5. Select mode (Core or Advanced)

## Customizing Presets

All presets can be customized:
- Adjust expertise level (Junior → Senior → Expert)
- Add domain-specific requirements
- Modify tone/style
- Add additional constraints
- Include industry-specific terminology
